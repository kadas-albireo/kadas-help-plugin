Substitutions¶

Usage¶

When you want to use an icon from KADAS in a rst document there is a big chance that there is allready a substitution defined in conf.py that can/should be used. When a new icon in QGIS is added it should be added first to conf.py before it is used in a rst document.

Below the icons and the Substitions defined in conf.py are given.
